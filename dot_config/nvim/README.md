# ab2vim

My [Neovim](https://github.com/neovim/neovim) settings.

## Installation

```sh
$ git clone https://github.com/Allianaab2m/ab2vim ~/.config
$ NVIM_APPNAME=ab2vim nvim
# or
$ export NVIM_APPNAME=ab2vim
$ nvim
```
